Description

Details

GMWwise is a cross-platform plugin written in C++ that allows GameMaker: Studio projects to take advantage of Wwise, a powerful audio middleware tool. GMWwise is fully documented, easy to integrate, and flexible enough to support a variety of projects.

Time Frame

GMWwise was initially started as an extension of a pre-existing DLL that had been abandoned for a number of years. It is currently maintained as an open-source project on GitHub.
