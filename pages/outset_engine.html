Description

Details

Outset Engine is a custom C++ game engine that supports 2D development for Windows, OS X, and Linux platforms. Outset Engine was designed to provide a complete asset creation pipeline that matched my desired workflow. It supports an external text editor for easy localization, a data/level editor for content changes that don't require binary recompilation, and a suite of debugging tools to assist with development.

Time Frame

Outset Engine was started in May of 2017 and is currently in development.
